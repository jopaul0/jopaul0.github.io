## Design e Concepção Visual

O design deste portfólio é fruto de um processo de **curadoria técnica e direção criativa**. O preview da interface (UI) foi gerado via **Claude (Anthropic)**, seguindo instruções detalhadas e conceitos que estabeleci com base em referências de alto nível que encontrei na web.

Em vez de seguir um modelo genérico, utilizei engenharia de prompt para fundir elementos de usabilidade, estética moderna e hierarquia visual, garantindo que o layout reflita meu perfil como desenvolvedor.

### 📍 Referências Inspiradoras
Para a construção desta interface, analisei e utilizei conceitos visuais presentes nos portfólios de:

* [Rakesh Potnuru](https://iamrakesh.com/) - Minimalismo e fluidez.
* [Thainan Prado](https://thainanprado.com.br/) - Apresentação de projetos e clareza.
* [Ruan Costa](https://www.ruancostadev.com.br/) - Performance e tipografia.
* [Parthh](https://parthh.in/) - Micro-interações e modernidade.

> **Nota de Desenvolvimento:** Embora a IA tenha auxiliado na prototipagem do visual, toda a **implementação técnica**, a arquitetura de estilos em **Sass**, a lógica de componentes em **React** e a tipagem em **TypeScript** foram desenvolvidas manualmente por mim, unindo design assistido com engenharia de software rigorosa.
